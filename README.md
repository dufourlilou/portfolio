Mon site portfolio et CV


Présentation du Projet: 

Le projet du TP etait de créer, optimiser et deployer un site internet avec portfolio et cv professionnel sur le serveur universitaire de l'Esine. Ce TP a pour but de montrer la maitrise et les compétences acquises lors du premier semestre (en ce qui concerne l'intégration, le JS ainsi que outils et architecture)

Outils et Architecture Web:

Outils utilisés: 
-Éditeur de code : Visual Studio Code
-Transfert de fichiers : avec FileZilla
_Fonctionnalité du serveur: avec eduVPN

Architecture du Projet:
-Frontend :HTML5, CSS et JavaScript.
-Structure : Construit avec séparation des ressources (CSS, Images, HTML).
-Déploiement :Hébergement sur serveur universitaire(IP : 10.56.8.58) 

Pages incluses
-Accueil :Présentation globale et accroche
-CV : Détail de non parcours académique et professionel
-Portfolio : Les projets réalisés
-Contact :Formulaire pour la prise de contact.
